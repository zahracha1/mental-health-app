# 🧠 Mental Health Disorder Prediction System

AI-powered psychiatric disorder classification system developed using Machine Learning and Streamlit.

---

##  Project Overview

This application predicts mental health disorders based on emotional, behavioral, and cognitive symptoms.

The AI model classifies patients into one of the following categories:

- Bipolar Type-1
- Bipolar Type-2
- Depression
- Normal

---

##  Technologies Used

- Python
- Streamlit
- Scikit-learn
- SVM (Support Vector Machine)
- Pandas
- NumPy

---

##  Machine Learning Model

The final deployed model is:

- Optimized SVM Classifier
- Feature Selection
- Standard Scaling
- Probability Prediction

## ▶ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
